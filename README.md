# Library

Library is a premium editorial website for independently written nonfiction book summaries designed to be read in about 15 minutes.

## Development

This repository uses a minimal Next.js App Router foundation with TypeScript and CSS custom properties for the design system.

Planned package manager: `pnpm`.

```bash
pnpm install
pnpm dev
pnpm lint
pnpm typecheck
pnpm build
```

The temporary design-system showcase is available at `/dev/design-system` after dependencies are installed and the development server is running. It is marked `noindex` and can be removed once a permanent documentation or story surface exists.
